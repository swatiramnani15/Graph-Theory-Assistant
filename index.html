Try the Chatbot

Purpose

This chatbot was developed as an educational assistant for students learning graph theory. It provides definitions, explanations, and algorithmic guidance on important topics such as paths, cycles, trees, connectivity, Euler tours, Hamiltonian graphs, shortest path algorithms, and matrix representations of graphs.


The goal is to create an accessible interactive tool that helps students explore mathematical concepts in graph theory in a conversational way.
